# Algotrading Line 2
Script to trade WDO Market in MetaTrader 5

-------
WARNING: You might lose money using this script
-------

How it works:

Candle Period: 2 minutes


Indicator: Moving Average (21 Periods - Exponential) and RSI (2 bonds: 58 and 38)


If the open candle price >= EMA and RSI >= 58 == BUY


If the open candle price <= EMA and RSI <= 38 == SELL


Take profit: 10 points


Stop Loss: 12 points
